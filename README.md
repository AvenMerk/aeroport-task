## Перед запуском проекта

В директории с проектом запустить:

### `npm i`

для установки node modules

## Для запуска проекта

В директории с проектом запустить:

### `npm start`

Открыть [http://localhost:3000](http://localhost:3000) для просмотра в браузере.
